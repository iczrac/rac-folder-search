# 🎉 RAC Folder Search - v1.0.2 更新说明

## ✨ 主要更新

### 1. 插件名称更改
- **旧名称**：Folder Search with Symlink Support
- **新名称**：RAC Folder Search ⭐
- **更简洁**：更容易记忆和搜索

### 2. 添加快捷键 ⌨️
- **Mac**：`Cmd+Alt+F`
- **Windows/Linux**：`Ctrl+Alt+F`
- **一键打开**：无需再通过命令面板

### 3. 版本号更新
- **版本**：1.0.1 → 1.0.2
- **包名**：folder-search-symlink → rac-folder-search

---

## 🚀 如何使用新版本

### 安装

1. **卸载旧版本**（如果已安装）
   - 打开扩展面板（`Cmd+Shift+X`）
   - 搜索 "folder-search" 或 "rac"
   - 点击卸载

2. **安装新版本**
   - 扩展面板 → `...` → "Install from VSIX..."
   - 选择 `rac-folder-search-1.0.2.vsix`
   - 重启 VS Code

### 使用快捷键

```
按 Cmd+Alt+F（Mac）或 Ctrl+Alt+F（Windows/Linux）
→ 输入案件编号
→ 选择文件夹
```

---

## ⌨️ 快捷键详情

### 为什么选择 `Cmd+Alt+F`？

✅ **不冲突**：
- `Cmd+P` - VS Code 文件搜索（保留）
- `Cmd+Shift+F` - VS Code 全局搜索（保留）
- `Cmd+F` - 当前文件搜索（保留）
- `Cmd+Alt+F` - RAC 文件夹搜索（新增）✨

✅ **好记**：F = Folder（文件夹）

✅ **方便**：单手可以按（左手）

### 自定义快捷键

如果你想使用其他快捷键：

1. 按 `Cmd+K Cmd+S` 打开快捷键设置
2. 搜索 "folder-search.search"
3. 点击铅笔图标修改
4. 按下你想要的快捷键组合

**推荐的替代快捷键**：
- `Cmd+Shift+O`（O = Open）
- `Cmd+K Cmd+F`（两步快捷键）
- `Cmd+Alt+P`（P = Project）

## 💡 使用技巧

### 1. 快捷键优先
- ✅ 使用 `Cmd+Alt+F` 比命令面板快 3 倍
- ✅ 养成习惯：需要搜索文件夹时，直接按快捷键

### 2. 搜索技巧
- 输入案件编号的任何部分
- 支持大小写不敏感
- 支持部分匹配

### 3. 刷新缓存
- 添加新案件后，使用命令面板刷新缓存
- `Cmd+Shift+P` → "Refresh Folder Index Cache"

## 🔧 配置建议

推荐配置（已优化）：

```json
{
  "fold-search.followSymlinks": true,      // 跟踪符号链接
  "fold-search.maxDepth": 2,               // 只索引第一层
  "fold-search.includeFiles": false,       // 只显示文件夹
  "fold-search.cacheExpiryMinutes": 10,    // 缓存 10 分钟
  "fold-search.maxResults": 5000           // 最多 5000 项
}
```
