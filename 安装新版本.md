# 🎉 安装 RAC Folder Search v1.0.2

## ✨ 新版本亮点

1. **新名称**：RAC Folder Search（更简洁）
2. **快捷键**：`Cmd+Alt+F`（Mac）/ `Ctrl+Alt+F`（Windows/Linux）
3. **一键搜索**：无需再通过命令面板

---

## 📦 安装步骤

### 1. 卸载旧版本（如果已安装）

```
1. 打开扩展面板：Cmd+Shift+X
2. 搜索 "folder-search" 或 "rac"
3. 点击"卸载"按钮
4. 重启 VS Code
```

### 2. 安装新版本

**方法 A：通过 UI**
```
1. 打开扩展面板：Cmd+Shift+X
2. 点击右上角 ... 菜单
3. 选择 "Install from VSIX..."
4. 选择文件：rac-folder-search-1.0.2.vsix
5. 重启 VS Code
```

**方法 B：通过命令行**
```bash
code --install-extension rac-folder-search-1.0.2.vsix
```

## ⌨️ 快捷键使用

### 主快捷键
| 功能 | Mac | Windows/Linux |
|------|-----|---------------|
| 搜索文件夹 | `Cmd+Alt+F` | `Ctrl+Alt+F` |

### 使用流程
```
1. 按 Cmd+Alt+F
2. 输入案件编号（如：C2CAR2021）
3. 选择文件夹，按 Enter
```

### 示例
```
Cmd+Alt+F → "C2CAR2021" → 找到 3 个案件
Cmd+Alt+F → "2023" → 找到所有 2023 年案件
Cmd+Alt+F → "D00003" → 找到包含 D00003 的案件
```

## ⚙️ 推荐配置

打开设置（`Cmd+,`），搜索 `fold-search`：

```json
{
  "fold-search.followSymlinks": true,      // 跟踪符号链接
  "fold-search.maxDepth": 2,               // 只索引第一层
  "fold-search.includeFiles": false,       // 只显示文件夹
  "fold-search.cacheExpiryMinutes": 10,    // 缓存 10 分钟
  "fold-search.maxResults": 5000           // 最多 5000 项
}
```

## 📞 问题排查

### 问题：快捷键不生效
**解决方案**：
1. 确认已重启 VS Code
2. 检查快捷键是否被其他扩展占用
3. 尝试自定义其他快捷键（`Cmd+K Cmd+S`）

### 问题：搜索不到文件夹
**解决方案**：
1. 检查 `followSymlinks` 是否为 `true`
2. 检查 `maxDepth` 设置（推荐 2）
3. 刷新缓存：`Cmd+Shift+P` → "Refresh Folder Index Cache"

### 问题：找不到扩展
**解决方案**：
1. 确认 .vsix 文件路径正确
2. 尝试通过命令行安装
3. 检查 VS Code 版本（需要 1.75.0+）
