## 🎯 使用步骤

### 立即使用（本地安装）

```bash
# 1. 在 VS Code 中安装
# 扩展面板 → ... → Install from VSIX → 选择 folder-search-symlink-1.0.1.vsix

# 2. 使用
# Cmd+Shift+P → Search Folders → 输入案件编号
```

### 发布到 Marketplace（可选）

```bash
# 1. 修改 package.json
# "publisher": "你的发布者名称"
# "repository.url": "你的 GitHub 地址"

# 2. 发布
vsce login 你的发布者名称
vsce publish
```

## 🔧 需要用户填写的信息

### 如果要发布到 Marketplace

在 `package.json` 中修改：

```json
{
  "publisher": "your-publisher-name",  // ⚠️ 改为你的发布者名称
  "repository": {
    "url": "https://github.com/your-username/folder-search-symlink"  // ⚠️ 改为你的仓库
  }
}
```

**如何获取发布者名称**：
1. 访问 https://marketplace.visualstudio.com/manage
2. 登录并创建发布者
3. 记下发布者 ID

**详细步骤**：查看 `PUBLISHER_GUIDE.md`
