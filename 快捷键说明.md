## ⌨️ 快捷键

### 主要快捷键

| 功能 | Mac | Windows/Linux |
|------|-----|---------------|
| **搜索文件夹** | `Cmd+Alt+F` | `Ctrl+Alt+F` |

### 其他方式

你也可以通过以下方式打开搜索：

1. **命令面板**：
   - 按 `Cmd+Shift+P`（Mac）或 `Ctrl+Shift+P`（Windows/Linux）
   - 输入 "RAC" 或 "Search Folders"
   - 选择 "RAC Folder Search: Search Folders"

2. **刷新缓存**：
   - 按 `Cmd+Shift+P`（Mac）或 `Ctrl+Shift+P`（Windows/Linux）
   - 输入 "Refresh"
   - 选择 "RAC Folder Search: Refresh Folder Index Cache"

## 🚀 使用流程

### 快速搜索案件文件夹

```
1. 按 Cmd+Alt+F（Mac）或 Ctrl+Alt+F（Windows/Linux）
2. 输入案件编号（如：C2CAR2021）
3. 选择文件夹，按 Enter 打开
```

### 示例

```
Cmd+Alt+F → 输入 "C2CAR2021" → 找到 3 个匹配的案件
Cmd+Alt+F → 输入 "2023" → 找到所有 2023 年的案件
Cmd+Alt+F → 输入 "D00003" → 找到包含 D00003 的案件
```

## 💡 为什么选择 `Cmd+Alt+F`？

### 优点
- ✅ **不冲突**：不会覆盖 VS Code 默认快捷键
- ✅ **好记**：F = Folder（文件夹）
- ✅ **方便**：单手可以按（左手）
- ✅ **通用**：Mac 和 Windows/Linux 都支持

### 避免的冲突
- ❌ `Cmd+P`：VS Code 默认的文件搜索
- ❌ `Cmd+Shift+F`：VS Code 默认的全局搜索
- ❌ `Cmd+F`：VS Code 默认的当前文件搜索

## ⚙️ 自定义快捷键（可选）

如果你想使用其他快捷键，可以在 VS Code 中自定义：

### 步骤

1. **打开快捷键设置**
   - Mac：`Cmd+K Cmd+S`
   - Windows/Linux：`Ctrl+K Ctrl+S`

2. **搜索命令**
   - 输入 "folder-search.search"

3. **修改快捷键**
   - 点击命令旁边的铅笔图标
   - 按下你想要的快捷键组合
   - 按 Enter 保存

### 推荐的替代快捷键

| 快捷键 | Mac | Windows/Linux | 说明 |
|--------|-----|---------------|------|
| 当前默认 | `Cmd+Alt+F` | `Ctrl+Alt+F` | 推荐使用 |
| 备选 1 | `Cmd+Shift+O` | `Ctrl+Shift+O` | O = Open |
| 备选 2 | `Cmd+K Cmd+F` | `Ctrl+K Ctrl+F` | 两步快捷键 |
| 备选 3 | `Cmd+Alt+P` | `Ctrl+Alt+P` | P = Project |

## 📦 安装新版本

### 方法 1：通过 VS Code UI

1. 打开 VS Code
2. 按 `Cmd+Shift+X` 打开扩展面板
3. 点击右上角的 `...` 菜单
4. 选择 "Install from VSIX..."
5. 选择 `rac-folder-search-1.0.2.vsix`
6. 重启 VS Code

### 方法 2：通过命令行

```bash
code --install-extension rac-folder-search-1.0.2.vsix
```
